Características de un CMS

  • Acceso web
  • Permite separar el contenido del diseño
    • Se puede cambiar el diseño del sitio Web sin que afecte al contenido. El contenido se adapta a los cambios realizados en diseño.
  • Las funcionalidades son provistas por módulos o complementos independientes ( “plugins”)
    • Los plugins se puede desarrollar a medida e incorporarlos al CMS.
      • Sitios Web especializados tendrán funciones específicas (Ej: Sitio e-commerce tendrá la gestión de los productos)
  • Rápida curva de aprendizaje
  • Sencilla gestión de contenidos y recursos
  • Interface de administración y gestión
  • Perfiles de usuario
  • Editor de contenidos (en línea)
  • Categorización de contenidos
  • Integración de complementos (plugins) y API de programación
  • Diseño visual personalizadle (temas, plantillas)
  • Gestión separada del contenido y del diseño
  • (Search Engine Optimization) SEO-friendly
