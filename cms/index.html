CMS

MOTIVACION:
  • Una empresa requiere un portal Web para realizar su publicidad. El sitio web considera varios contenidos. El sitio no debe ser algo estático sino algo que crezca y evoluciones con el tiempo para crear nuevos contenidos o actualizar los existentes. Por ejemplo, la página de testimonios de clientes deberá actualizarse frecuentemente Los contenidos irán creciendo según demanda, por tanto se deberá poder crear páginas web con cada contenido variado. Las personas que ingresarán los contenidos no disponen de conocimiento tecnológico.
