Instalación

Pre-requisitos

  • PHP version 7.4 or greater.
  • MySQL version 5.6 or greater OR MariaDB version 10.1 or greater.
  • HTTPS support

Instalación local de Wordpress

Práctica:
  1. Descargar Wordpress (https://es-ec.wordpress.org/download/)
  2. Descomprimir wordpress en directorio raiz (htdocs del servidor apache instalado en máquina local)
  3. Crear base de datos wpsitio usando phpmyAdmin (utf8_spanish_ci)
    1. Crear usuario de bdd: wpusuario / 48suEk96667erPF
  4. Desplegar el sitio: Ir a localhost/wpsitio e iniciar instalación
    1. Titulo Sitio: AppForAll
    2. Usuario admin: wpsitio / adminEoPir863
Una vez el sitiohaya sido instalado. Puedes ingresar a la administración de tu sitio wprdPress a través de: http://localhost/wpsitio/wp-login.php
