Instalación en host

La puesta en producción de un sitio creado en Wordpress se puede resumir en los siguientes pasos:

Pre-requisitos:

  1. Un programa cliente de FTP (p.ej., FileZilla)
  2. Un dominio (p.ej., www.midominio.com) ; varias empresas ofrecen dominios (p. ej., GoDaddy [https://www.godaddy.com/es].
  3. Un hosting propio
    • En EEUU:
      • bluehost
      • webempresa
      • HostGator 
      • Lista: https://elmejorhosting.net/top10-hosting-estados-unidos/ .
    • En Ecuador:
      • EcuaHosintg (https://ecuahosting.net/)

Instalación:

  1. En computadora local:
    1. Ingresar a administración de wordpress y cambiar el dominio del sitio con el dominio obtenido en punto 2 de Pre-requisitos 
    2. Obtener respaldo de la BDD (hacer un volcado de la base de datos). Se debe obtener un fichero con extensión sql.
    3. Obtener respaldo del sitio
  2. En servidor de hosting
    1. Ingresar a administrador de hosting (Cpanel)
    2. Acceder a phpMyAdmin y crear una base de datos vacía.
    3. Crear un usuario específico para la base de datos vacía.
    4. Importar el fichero indicado en 1.1 , sobre la base de datos vacía.
    5. Copiar el respaldo 1.2 en el directorio htdocs del servidor.
    6. Probar que el sitio está funcionando.- Ingresar con un navegador al dominio (Ej., www.midominio.com)
