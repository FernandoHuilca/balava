LAS FRACCIONES

Para iniciar nuestras clases del Noveno año es necesario que domines el manejo de operaciones con números enteros y fraccionarios, ya que durante el año lectivo realizaremos cálculos con números racionales, irracionales y reales, por lo tanto ustedes deben dominar estas destrezas en la solución de problemas.


Vamos a practicar algunos ejercicios para fortalecer esta destreza.


Llamamos números racionales al conjunto formado por todos los números enteros y todos los fraccionarios, se le designa por Q y se lo denomina conjunto de los números
racionales.

Número racional es el que se puede expresar
como cociente de dos números enteros, es decir, en forma de fracción.

las fracciones se conforman por numerador y denominador.

COMPARACIÓN DE FRACCIONES

Para comparar dos números racionales, se deben tener en cuenta varios criterios:

• Todo número racional positivo es mayor que cualquier racional negativo.

• Todo número racional negativo es menor que 0.

• Si dos números racionales positivos tienen igual denominador, es menor el que tiene menor numerador.
Por ejemplo: 3/5 es menor que 4/5

• Si dos números racionales positivos tienen el mismo numerador, es menor el
que tiene mayor denominador.
Por ejemplo: 3/7 es menor que 3/4

• Si dos números racionales tienen distinto denominador, se debe buscar una fracción equivalente a cada una de las fracciones dadas, con el mismo denominador, y compararlas teniendo en cuenta los criterios anteriores.

ADICIÓN DE NÚMEROS RACIONALES CON NÚMEROS FRACCIONARIOS

CASO 1:
Para sumar dos números racionales con el mismo denominador, se suman los numeradores y se mantiene el mismo denominador.

CASO 2:
Para sumar dos números racionales con diferente denominador, se buscan fracciones equivalentes a los números racionales dados, que tengan el mismo denominador; luego se adicionan las fracciones equivalentes obtenidas como en el caso anterior.

Sustracción de números racionales en expresión fraccionaria

CASO 1:
Para sustraer números racionales con igual denominador, se restan los numeradores y se deja el mismo denominador.

CASO 2:
Para sustraer racionales con diferente denominador, primero se hallan fracciones equivalentes a los números racionales dados que tengan el mismo denominador; luego, se procede como en el caso anterior.

MULTIPLICACIÓN Y DIVISIÓN DE FRACCIONES

Multiplicación:
Para multiplicar tres o más números racionales en su expresión fraccionaria, se multiplican los numeradores entre sí y los denominadores entre sí.

Antes de efectuar productos de números racionales, conviene simplificar al máximo

División:
Para dividir dos números racionales, se multiplica el dividendo por el inverso multiplicativo del divisor.(recuerdas que la segunda fracción la invertiamos y la división se convertía en multiplicación)

OPERACIONES COMBINADAS

Para desarrollar las operaciones combinadas, es decir doende encuentres adición, sustracción, multiplicación y división debes considerar los siguientes criterios:

1. Convertir los números mixtos o decimales
2. Calculamos potencias y raíces
3. Resolvemos los paréntesis de más "adentro"
4. Desarrollamos las multiplicaciones y divisiones
5. Resolvemos las adiciones y sustracciones
