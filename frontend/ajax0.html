Ajax

Load (cargar Archivo)
load(url [,datos,][,función]) cargar un fichero siguiendo el principio que aplica AJAX.
Consulta por GET
$get(url [,datos,][,función][,tipo]) forma abreviada para consulta AJAX con GET
Consulta por POST
$post(url [,datos,][,función][,tipo]) forma abreviada para consulta AJAX con POST
Consulta General
ajax(options)

Consulta General

$.ajax({
url: “test.html”,
success: function (data){
    $(“#resultado”).html(data);
    $().log(“Terminado”);
},
})

Métodos y atributos posibles:

  • url = cadena de caracteres que contiene la dirección la consulta
  • type = (Opcional) cadena de caracteres que define el método HTTP utilizado para la consulta. Por defecto es GET
  • dataType = (Opcional) cadena de caracteres que especifica el formato d los datos que se enviarán por el servidor (xml, html, json, script)
  • timeout = (opcional) número de milisegundos tras el cual la consulta se considera fallida.
  • beforesend = (opcional) función que se debe ejecutar antes del evento de la consulta
  • error = (opcional) función que se debe ejecutar sin falla la consulta
  • success (opcional) función que se invoca si la consulta se ejecuta con éxito
  • complete = (opcional) función que se tiene que ejecutar cuando la consulta termina.
  • data = (opcional) datos que se envían al servidor. Objeto debe estar en forma de pares de clave/valor
  • contentType = (opcional) cadena de caracteres que contiene el MIME de los datos cuando se envían al servidor. Por defecto: aplicación/x-www-form-urlencoded
  • async = (opcional) valor booleano que indica si la consulta se deve efectuar síncrona o asíncronamente. Por defecto true (asíncrona).
  • statuscode = (opcional) permite llamar a una función cuando se produce un error con un código especifico.
