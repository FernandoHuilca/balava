Iniciación

  • Todos los scripts jQuery inician con:
jQuery(function(){
    //contenido que se ejecuta cuando se carga el documento
});
  • Usando “$” como Alias de jQuery 
$(function(){
    //contenido que se ejecuta cuando se carga el documento
});
  • Bajo este modelo, para utilizar los elementos del DOM una vez ya estén cargados, el script jQuery empieza por:
<script>
$(document).ready(function(){ 
    //codigo jQuery
}):
</script>
En este último ejemplo, la línea 2 hace que se cargue el árbol DOM del documento en un objeto jQuery. Esta línea se puede traducir en:
 “Crear un objeto jQuery($) a partir del documento (document), cuando esté preparado (ready)”
La línea 2 sería equivalente a lo que en JavaScript tradicional se escribe como: window.onload = function()
