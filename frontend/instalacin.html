Instalación

  • Local
    1. Acceder a https://jquery.com/download/
    2. Escoger la versión
      • Existen versiones comprimidas y no comprimidas:
      • Versión no comprimida recomendable para “desarrollo”
      • Versión comprimida recomendable para “producción”
    3. Abrir fichero en el Navegador
    4. Guardar fichero (“Guardar como …” ) en el directorio del proyecto.
  • Por CDN
    • <script src=http://code.jquery.com/jquery-numero de versión.min.js></script>
  • API de Google
    • <script type='text/javascript' src="//ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
