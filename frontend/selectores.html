Selectores

JQuery

Comentario

Correspondencia con JS tradicional

$("*")

Seleccionar todos los elementos

$(“.clase”)

Seleccionar por Clase

document.getElementByClassName(“clase”)

$("elemento")

Seleccionar por Tag

document.getElementByTagName(“tag”)

$("#id")

Seleccionar por ID

document.getElementById(“id”)

Ejemplos

Mira algunos ejemplos de selectores. Fuente: https://uniwebsidad.com/libros/fundamentos-jquery/capitulo-3/seleccion-de-elementos

Selección de elementos en base a su id

$('#myId'); // notar que los IDs deben ser únicos por página

Selección de elementos en base al nombre de clase

$('div.myClass'); // si se especifica el tipo de elemento,
                  // se mejora el rendimiento de la selección
Selección de elementos por su atributo
$(“div[id]”); //Selecciona elementos que tienen un atributo id

Selección de elementos por su atributo con valor

$('input[name=first_name]'); // tenga cuidado, que puede ser muy lento

Selección de elementos en forma de selector CSS

$('#contents ul.people li');
