Instrucciones

Video de instrucciones: David Bueno Vallejo

  1. Acceder a repositorio Git del proyecto: AQUI
  2. Navegar hacia abajo hasta encontrar "Installation"
  3. Copiar la url de "Install from the uptdate site" (https://takezoe.github.io/amateras-update-site/)
  4. Ir a Eclipse > Help > Install new software
  5. "Agregar site", pegando la url en el paso 3
  6. Seleccionar las opciones de plugin
  7. Instalar

Nota: For users who use Eclipse on Java 16 or later, add the following to eclipse.ini : --add-opens=java.desktop/java.beans=ALL-UNNAMED
